# Sales Analyzer Dashboard (Power BI)

## Overview

The Sales Analyzer Dashboard is an end-to-end business intelligence project built using Power BI. It is designed to provide actionable insights into sales performance, customer behavior, and revenue trends. This project demonstrates strong capabilities in data modeling, DAX, and interactive dashboard design, making it suitable for real-world business scenarios.

## Objective

The goal of this project is to transform raw sales data into meaningful insights that help stakeholders:

* Track sales performance
* Identify high-value customers
* Analyze product and regional trends
* Support data-driven decision-making

## Key Features

* Interactive dashboard with dynamic filtering
* KPI tracking for revenue, customers, and sales growth
* Customer segmentation based on purchase behavior
* Sales trend analysis over time
* Region-wise and product-wise performance breakdown
* Identification of high-value customers (Sales > 1000)

## Tools and Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization

## Data Model

The project uses a star schema data model consisting of:

* Fact Table: Sales data containing transaction-level details
* Dimension Tables:

  * Customer Dimension
  * Product Dimension
  * Date Dimension
  * Region/Location Dimension

Relationships are optimized to ensure efficient querying and accurate reporting.

## Key DAX Measures

Some important calculations implemented in this project include:

* Total Sales
* Distinct Customer Count
* High-Value Customers (Sales > 1000)
* Sales Growth Percentage
* Time Intelligence Measures (YTD, MTD)

## Dashboard Highlights

* Clean and professional layout

## Business Insights

1. Download the .pbix file

3. Interact with the dashboard using filters and slicers
4. Explore insights across different dimensions
